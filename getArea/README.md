```
micro new micro/getArea --namespace=go.micro --type=srv
```

## Configuration

- FQDN: go.micro.srv.getArea

## Dependencies

Micro services depend on service discovery. The default is consul.

```
# install consul
brew install consul

# run consul
consul agent -dev
```

## Usage

A Makefile is included for convenience

Build the binary

```
make build
```

Run the service
```
./getArea-srv
```

Build a docker image
```
make docker
```
